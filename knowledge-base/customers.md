# Customer Notes

This file tracks preferences, history, and context for recurring customers.
The call processor automatically creates/updates FreshBooks entries — this file captures
the nuance that doesn't fit in a CRM field.

## Template for a customer entry

### [Customer Name]
- **Machine(s):** [year, make, model]
- **History:** [summary of past work]
- **Preferences:** [how they like to be contacted, any specific requests]
- **Open items:** [anything waiting on parts, follow-up, etc.]
- **Red lines:** [anything to never do / always confirm first]
- **Source:** call transcript [date] / FreshBooks invoice [#]
